## Template

This is a template from [vite-react-tailwind-starter`](https://github.com/sozonome/vite-react-tailwind-starter)
```
pnpm i
```

Then, run the development server:

```bash
pnpm dev
```

## Deployment

- build command: `pnpm build`
- output directory: `dist`
